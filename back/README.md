# Prueba tecnica para Carsales

## Requisitos previos
- [.NET 8 SDK](https://microsoft.com)
- visual studio o cualquier editor de código compatible con .NET

## Instalación
```bash
git clone https://github.com/ClaudioVargas/PruebaTecnicaCarsales.git
cd PruebaTecnicaCarsales/back
dotnet restore
```

## Ejecución
```bash
dotnet run
```

## Documentación
- una vez levantado el proyecto con dotnet run , el backend estará disponible en http://localhost:5074
- para probar lo puedes hacer con el archivo back.http que se encuentra en la raíz del proyecto. 
- una vez levantado debes presionar Enviar solicitud abajo del comentario ###
Consulte la [documentación oficial de .NET](https://learn.microsoft.com/dotnet) para más detalles.
